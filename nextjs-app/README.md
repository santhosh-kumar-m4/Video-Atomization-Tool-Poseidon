# Video Atomization Tool - Next.js App

This is the Next.js application for the Video Atomization Tool.

**Note:** For complete setup instructions, architecture details, and documentation, please see the main [README.md](../README.md) in the project root.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file (see main README for details)

3. Initialize database:
```bash
npm run db:init
```

4. Start development server:
```bash
npm run dev
```

The application will be available at `http://localhost:4201`

For full documentation, see the [main README](../README.md).
